本着打不过就加入的原则，做了这期分享。用好这些AI工具能够提升工作效率，但是也不能过渡依赖，否则就真的成为“字节搬运工”了！

ChatGPT ​

ChatGPT（全名：Chat Generative Pre-trained Transformer），美国 OpenAI 研发的聊天机器人程序。ChatGPT 是人工智能技术驱动的自然语言处理工具，它能够通过理解和学习人类的语言来进行对话，还能根据聊天的上下文进行互动，真正像人类一样来聊天交流，甚至能完成撰写邮件、视频脚本、文案、翻译、代码，写论文等任务。

（因为 IP 被封，我已经用不了了 😅 ，只能调调 API）

官网地址：https://chat.openai.com/chat

开发平台：https://openai.com/

🚫 只有 Plus 用户才能使用 GPT4

🚫 受地区限制，需要魔法 🪄

Monica ​

Monica 是由 ChatGPT API 提供支持的 Chrome 扩展程序，旨在成为您的个人 AI 助手，实现轻松聊天和文案编写。

官网地址：https://monica.im/

✅ 添加到 chrome 插件

✅ 添加到 Edge 插件

✅ 支持升级到付费版本

🚫 每天 40 次免费问答

🚫 一共 5 次网页阅读理解机会

Poe ​

Poe 是 AI 机器人集合 Sage、Claude、ChatGPT、Dragonfly 。poe.com，这个网站在一个界面集成了包括 ChatGPT 在内的四个不同的 AI 机器人，响应速度挺快。目前无需付费。

官网地址：https://poe.com/

✅ 支持多种聊天模型：Sage、，Claude+、GPT-3.5、GPT-4 等

✅ 支持创建自己的聊天机器人（通过 prompt 指定 bot 的角色，在指定领域问答）

🚫 需要魔法 🪄

🚫 GPT-4 每天 1 次免费机会

🚫 Claude+每天 3 次免费机会

官网地址：https://www.anthropic.com/

在 slack 上使用： https://www.anthropic.com/claude-in-slack

✅ 无需魔法

✅ 无次数限制

文心一言是百度公司开发的人工智能语言模型，我可以协助您完成范围广泛的任务并提供有关各种主题的信息。支持文本对话，文生图。

Auto-GPT ​

Auto-GPT 是一个实验性的开源应用程序，展示了 GPT-4 语言模型的能力。这个程序由 GPT-4 驱动，通过链接 LLM“思想”，来自主地实现您设定的任何目标。作为 GPT-4 完全自主运行的首批示例之一，Auto-GPT 推动了 AI 所能实现的界限。

github 地址：https://github.com/Significant-Gravitas/Auto-GPT

✅ 自问自答，有了点“贾维斯”的影子

🚫 需要本地部署

（我也就是部署之后体验了一天，目前对我没太大用处 🤷）

AgentGPT ​

AgentGPT是一个在浏览器中运行的任务驱动型自主AI代理。通过利用先进的AI模型，如OpenAI的GPT-4语言模型，这些代理不仅能够执行任务，还可以根据结果生成新的任务并动态优先考虑它们。

官网地址：https://agentgpt.reworkd.ai/

✅ 在线体验AutoGPT的魔力

🚫 需要提供openai key
