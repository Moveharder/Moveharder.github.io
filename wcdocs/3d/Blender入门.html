快捷键 ​

物体操作 ​

  • Tab 切换物体模式/编辑模式
  • Shift + A 新建物体面板
  • Shift + D 复制物体
  • Alt + D 关联复制（这样在草走其中一个物体时，被复制物体会同步变化，更加高效）
  • Shift + 左键点击物体，可多选物体(批量复制时用到了)
  • Shift + 右键物体，放置游标位置
  • / 进入隔离模式，左上角下拉找到“编辑模式”，对点、线、面(快捷键：1、2、3)进行操作
  • X 删除物体
  • G 旋转，G + X|Y|Z 指定某个轴移动
  • R 旋转，R + X|Y|Z 指定某个轴旋转
  • S 缩放，S + X|Y|Z 指定某个轴缩放
  • Q 展示收藏的快捷操作（右键某个操作可以加入到收藏中）
  • ~ 出现视图的雷达圆盘
  • RR（按两次 R，自由旋转）
  • Alt + R 复位旋转
  • N 打开侧边栏工具
  • M 选中物体后，按M键可以新建集合（给模型分组管理）
  • Ctrl + A 设置物体的缩放（缩放后设置倒角效果异常时可用此修复）
  • Ctrl + L 选择“材质关联”：选中其他未设置材质的物体，最后选中已经设置材质的物体，选择“关联材质”，会为所选材质赋予相同的材质。
  • Ctrl + L 选择“复制修改器”：可以设置已有物体的修改器到其他物体上，类似“材质关联”的操作。
  • Ctrl + 2 快速“细分曲面”，2表示2级细分。
  • Ctrl + R 环切：可用于给一个不同口径的柱体，从上往下增加圆滑过渡。(滚动滚轮可调整环切数量)

制作动画（时间-变量-属性） ​

  • I 插入关键帧（先选中物体，设置好时间轴位置，设置好物体的属性值）

高效设置 ​

  • 勾选 Gizmo 里的[移动选项]

  • 勾选 Shading 里的[Cavity]

高级操作（基本都在「编辑模式」下） ​

  • 挤出工具，左边工具里。能够实现边的挤出，比如给地板向上拉一个背景帷幕
  • 添加倒角：选中物体，右侧[扳手 🔧]图标，可以设置[数量][段数]，调整倒角弧度、圆滑程度
  • 材质关联：选中无材质物体，最后选择已有材质的物体，按 Ctrl+L，出现选项里选择材质
  • 高对比度：右侧-渲染属性-色彩管理-胶片效果-下拉选择「high contrast」
  • 挤出：线-->选中某条线，按E（+XYZ指定方向）挤出，可以延展一个面。如果选中面，E挤出这个面，此时「按S」可以缩放这个面（柱体制作铅笔头）
  • 内插面：选中某个面，按 I 键（按住Shift微调）。
  • 特定位置倒角：选中某些边，Ctrl+B，拖动可以设置倒角大小，此时滚轮可控制段数
  • 选中循环边：Alt+点击某个边
  • 桥接循环边：选中上下两条循环边，右键-->桥接循环边：制作一个圆筒
